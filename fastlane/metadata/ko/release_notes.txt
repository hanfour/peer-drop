PeerDrop 3.0 — 종단간 암호화 통신 + 펫 진화 완전판

종단간 암호화 원격 통신
- Double Ratchet 암호화 프로토콜: 네트워크 간 안전한 통신
- 신뢰 연락처 관리: 초대, 수락, 검증
- 암호화 세션 자동 영속화, 재시작 후 원활한 복원
- 보안 대시보드: 모든 암호화 연결 상태 확인

펫 시스템 완전판
- 10종 체형 × 2단계 × 20종 애니메이션
- 펫 전용 탭: 프로필, 유전자, 성격, 통계
- 음식 인벤토리: 드래그로 급여, 펫이 쫓아가서 먹음
- 급여→소화→배변 전체 사이클
- 성격 기반 행동: 에너지가 활동량에, 장난기가 등반에 영향
- Dynamic Island + 홈/잠금 화면 위젯
- 매일 로그인 시 음식 보충, 연결 시 음식 획득

연결 개선
- Relay 연결 속도 최적화
- PIN 인증 코드가 수신되지 않는 문제 수정
