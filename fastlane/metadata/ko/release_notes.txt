PeerDrop 2.1.0 — 펫 진화 대개편

새로운 펫 시스템
- 10종 체형: 고양이, 강아지, 토끼, 새, 개구리, 곰, 드래곤, 문어, 유령, 슬라임
- 유아기와 성장기 2단계, 각 20종 애니메이션
- 16×16 픽셀 아트 스프라이트, 8가지 색상 팔레트

물리 인터랙션
- 펫이 화면에서 걷고, 벽을 오르고, 천장에 매달림
- 드래그 앤 던지기로 중력과 바운스 물리
- 쓰다듬기 제스처로 하트와 XP 획득

음식 시스템
- 아이템 인벤토리: 주먹밥, 작은 생선, 사과
- 음식을 화면에 드래그하면 펫이 쫓아가서 먹음
- 매일 로그인 시 음식 보충, 연결 시에도 음식 획득

진화와 성장
- XP + 시간으로 다음 단계로 진화
- 진화 시 햅틱 피드백과 플래시 효과
- 부화 시 펫 이름 짓기

Dynamic Island 및 위젯
- Dynamic Island에서 펫 상태 표시
- 홈 화면 및 잠금 화면 위젯

연결 수정
- Relay 연결 시 참가자가 PIN 인증 코드를 받지 못하는 문제 수정
- Relay 연결 속도 최적화
