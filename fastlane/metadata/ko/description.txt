PeerDrop은 로컬 네트워크에서 기기 간에 직접 파일 공유, 채팅, 음성 통화를 할 수 있는 앱입니다. 인터넷, 클라우드, 계정 등록이 필요 없습니다.

주요 기능

- 직접 파일 전송: 사진, 동영상, 문서 등 모든 파일을 근처 기기로 직접 전송합니다. 파일은 청크로 분할되며 SHA-256 해시로 무결성을 검증합니다.

- 암호화 채팅: 텍스트 메시지, 사진, 동영상, 음성 메시지를 종단간 암호화로 전송합니다. 모든 채팅 기록은 로컬에 저장되며 AES-256-GCM으로 암호화됩니다.

- 음성 통화: WebRTC를 통한 선명한 음성 통화. iOS CallKit과 완전히 통합되어 네이티브 통화 경험을 제공합니다.

- 기기 검색: Bonjour를 통해 같은 Wi-Fi 네트워크의 근처 기기를 자동 검색합니다. IP 주소로 수동 연결도 가능합니다.

- 다중 연결: 최대 5대의 기기에 동시 연결 가능. 각 연결에는 독립적인 채팅, 파일 전송, 음성 통화 기능이 있습니다.

- 기기 라이브러리: 자주 연결하는 기기를 저장하여 빠르게 재연결. 그룹으로 정리할 수 있습니다.

- 프라이버시 우선: 서버 없음, 계정 없음, 추적 없음. 모든 것이 기기 간에 완결됩니다.

사용 방법

1. 같은 Wi-Fi에 연결된 두 기기에서 PeerDrop을 엽니다
2. 기기가 자동으로 검색됩니다
3. 탭하여 연결 — 수신 측에서 연결을 승인합니다
4. 파일 공유, 채팅 또는 음성 통화를 시작합니다

보안

- 임시 P-256 키 쌍을 사용한 TLS 암호화 연결
- AES-256-GCM으로 채팅 데이터 암호화 저장
- 인증서 지문 검증 (첫 사용 시 신뢰)
- SHA-256 파일 무결성 검증
- 데이터는 로컬 네트워크 밖으로 나가지 않습니다
