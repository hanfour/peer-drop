이제 PeerDrop이 Mac에 등장합니다. iPhone·iPad 또는 다른 Mac과 직접 파일 공유·채팅·음성 통화 ― 인터넷, 클라우드, 계정 모두 필요 없습니다.

MAC 네이티브 경험

- 메뉴 막대 항목:상태, 주변 기기 목록, 빠른 파일 전송. 상대 행에 파일을 직접 끌어다 놓으면 전송.
- 멀티 윈도우 채팅:상대마다 별도 창. ⌘1로 최근 채팅으로 이동.
- Dock + Finder 드래그:Finder에서 파일을 PeerDrop의 Dock 아이콘이나 메인 창에 끌면, 상대를 고르는 확인 시트가 나타납니다.
- Spaces 호환 통화:착신 시 떠 있는 창이 Spaces나 전체 화면 앱을 전환해도 따라옵니다. 통화 중 창도 항상 최상위에 유지됩니다.

주요 기능

- 직접 파일 전송:사진, 동영상, 문서, 어떤 파일이든 주변 기기로 직접 전송. 청크 단위 업로드와 SHA-256 해시로 무결성 검증.

- 암호화 채팅:텍스트, 사진, 동영상, 음성 메시지를 종단간 암호화(X3DH + Double Ratchet). 기록은 기기 안에 AES-256-GCM으로 암호화되어 저장.

- 음성 통화:WebRTC 고품질 통화. iPhone은 CallKit, Mac은 자체 떠 있는 창을 사용해 화면을 가로채지 않습니다.

- 기기 검색:Bonjour로 같은 Wi-Fi의 기기를 자동 검색. IP로 수동 연결도 가능.

- 다중 연결:최대 5대까지 동시 연결. 연결마다 채팅, 파일, 통화가 독립적으로 동작.

- 기기 라이브러리:자주 쓰는 기기를 저장하고 그룹화해 한 번에 재연결.

- 프라이버시 우선:서버 없음, 계정 없음, 추적 없음. 데이터는 기기 사이에만 흐릅니다. TLS 암호화와 TOFU(첫 연결 신뢰)로 인증서 지문 검증.

사용 방법

1. Mac(macOS 14 이상)과 iPhone 또는 iPad에서 PeerDrop을 실행하고 같은 Wi-Fi에 연결
2. 기기가 서로를 자동으로 찾습니다
3. 탭하여 연결 ― 받는 기기에서 SAS(짧은 인증 코드)를 확인하고 승인
4. 파일 공유, 채팅, 음성 통화 시작

보안

- 일회용 P-256 키 기반 TLS 암호화 연결
- 채팅 기록은 AES-256-GCM으로 암호화 저장
- 첫 연결 시 인증서 지문 검증(TOFU)
- 파일 SHA-256 무결성 확인
- 데이터는 로컬 네트워크 밖으로 나가지 않습니다
- App Sandbox 사용. PeerDrop이 실제로 필요한 권한만.

6.0 새로운 기능

첫 Mac 버전입니다. iPhone/iPad 6.0은 같은 코드를 Swift Package(PeerDropKit)로 재구성한 것이며, 사용자 경험은 동일합니다.
