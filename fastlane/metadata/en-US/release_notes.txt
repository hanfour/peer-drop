PeerDrop 1.3.0 — Quality & Usability Improvements:

- Fixed reconnection failures when connecting to the same device twice
- Received files now saved to Documents/PeerDrop, visible in Files app
- Duplicate file names automatically get a sequence number
- Device stays discoverable via Bluetooth in background
- Local connections preserved when app is suspended
- Swipe to edit or delete manual connections
- QR code for easy device-to-device connection setup
- Deep link support: peerdrop://connect/HOST:PORT
- Manual peers auto-removed after repeated connection failures
