### v4.0.1 — Hotfix dispatched 2026-05-07

v4.0.0 build 1 went live 2026-05-07 17:32 Taipei (phased rollout Day 1).
User reported: pet egg disappeared after auto-update.

Root cause: v4.0 ship missed egg-stage sprite assets; PetRendererV3
fallback chain hits cat-tabby placeholder for any .egg-stage pet.

stage entirely. PetLevel decoder maps legacy rawValue 1 → .baby
