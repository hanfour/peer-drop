## What's new in v5.5.0 → v5.5.1

**More animated pet species.** v5.5.0 unlocked a wider range of virtual-pet species. Several of them shipped with still-frame sprites only. This patch adds the proper multi-frame walk + idle animations for 40 of those species×stages — dragons, slimes, totoro, unicorns, otters, and the available life-stages of birds, frogs and octopuses now move (walk cycle + gentle idle) the same way the original cat/dog/bear pets already did.

That is the entire change set: bundled sprite-animation data. There is no new code path, no new UI, and no change to how the pet, messaging, file transfer, or any other feature behaves. A pet that previously appeared frozen now animates; nothing else differs.

## What didn't change

- End-to-end encryption (X3DH + Double Ratchet), local Wi-Fi + relay transports, SAS pairing — unchanged.
- Cross-device pet sync (added in v5.5.0) — unchanged.
- File transfer, voice calls, screenshot mode — unchanged.
- Tip Jar IAPs (tip.small / tip.medium / tip.large), Privacy Manifest (no new data types collected), age rating 4+, supply regions — unchanged.
- No new entitlements, no new permission prompts.

## How to verify

1. Install v5.5.1. Open the app and tap the "Pet" tab — the virtual pet renders and animates.
2. The floating pet on the main screen walks around with a multi-frame walk cycle and a subtle idle bob, regardless of which species the pet is.
