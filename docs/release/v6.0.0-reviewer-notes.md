# v6.0.0 — App Store Reviewer Notes (iOS + macOS)

**TL;DR:** PeerDrop comes to Mac. The macOS app is a peer of the existing iPhone/iPad app: bidirectional voice calling, file transfer, secure pairing, multi-window chat. iOS v6.0.0 itself is an internal codebase modularisation (SPM rebuild) with zero user-facing change. Both platforms share the same `v6.0.0` version per the cross-platform ship cycle. No new data collection.

## What's new in v6.0.0

**macOS app launches (com.hanfour.peerdrop.mac).** This is the first Mac binary. Feature parity with iOS for the core P2P flows:

- Peer discovery over local Wi-Fi (Bonjour) and Bluetooth, plus optional cross-network relay
- SAS-based device pairing with the existing iPhone/iPad app
- End-to-end encrypted chat (X3DH + Double Ratchet — same as iOS)
- File transfer via drag-and-drop (Dock icon, main window, per-peer rows in the menu bar)
- Bidirectional voice calls with iPhone

**Mac voice calling mechanism (differs from iOS):**

- macOS uses a custom floating window for incoming-call UI (CallKit is iOS-only). The window appears top-right of the active screen, shows the caller's name + accept/decline buttons, and stays visible across Spaces.
- Apple Push Notifications wake the app when sleeping — alert push, not VoIP push (PushKit is iOS-only). Microphone permission is requested on first call attempt via the standard `NSMicrophoneUsageDescription` prompt.
- Do Not Disturb compatibility: the incoming-call window still appears, but the ringtone is muted when system Notifications are silenced. (macOS 14 does not expose Focus mode state to third-party apps; per-Focus configurations are not directly readable.)
- Self-drawn incoming-call window precedent: FaceTime, Slack, Discord, WhatsApp Desktop all use custom windows on macOS. We do not draw lock-screen UI or simulate system trust elevation.

**iOS v6.0.0 is an internal SPM rebuild.** The codebase moved into a Swift Package (`PeerDropKit`) with six product modules (Platform / Protocol / Security / Pet / Transport / Core). There is no user-facing change — identical UI, identical behavior, identical entitlements. The version bump ships in lockstep with the Mac release.

## Sandbox + entitlements (Mac)

- `com.apple.security.app-sandbox` — required for Mac App Store
- `com.apple.security.network.client` + `com.apple.security.network.server` — peer-to-peer connections + relay
- `com.apple.developer.networking.multicast` — Bonjour discovery on local Wi-Fi
- `com.apple.security.device.bluetooth` — peer discovery via BLE
- `com.apple.security.device.audio-input` — voice call microphone capture
- `com.apple.security.files.user-selected.read-write` — files the user explicitly drops or picks
- `com.apple.security.files.downloads.read-write` — inbox + history caches
- `aps-environment` — APNs alert push for incoming-call wake

No `com.apple.security.files.all` (no full disk access), no Apple Events, no system extensions.

## How to verify (Mac voice)

1. Install v6.0.0 on a Mac (macOS 14+) and an iPhone (linked App Store record).
2. On both devices: tap "Pair Device" and confirm the SAS code matches. Same pairing UX as iPhone↔iPhone.
3. iPhone → Mac: tap the phone icon on the Mac peer row. Mac shows the floating panel; tap Accept; audio should be established within 2-3 seconds.
4. Mac → iPhone: open chat with the iPhone peer, tap the phone icon. iPhone shows the standard CallKit incoming UI.
5. Drag a file from Finder onto the menu bar PeerDrop icon → confirmation sheet → peer picker → send.

## What didn't change (iOS)

- End-to-end encryption fundamentals (X3DH + Double Ratchet) — unchanged.
- Local Wi-Fi transport and SAS pairing sheet — unchanged.
- File transfer, voice call (PushKit + CallKit), screenshot mode — unchanged.
- Push notifications, App Attest device-token authentication — unchanged.
- Tip Jar IAPs (tip.small / tip.medium / tip.large) — unchanged on iOS; Mac IAPs attached separately to the Mac binary.
- Privacy Manifest (`PrivacyInfo.xcprivacy`) — unchanged, no new data types collected.
- Age rating (4+), supply regions (174 countries, China excluded) — unchanged.
- Pet animations, rarity tiers, species catalog — unchanged.
