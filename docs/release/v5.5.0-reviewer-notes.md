# v5.5.0 — App Store Reviewer Notes

**TL;DR:** Feature + reliability release. Adds optional iCloud sync of the virtual pet across the user's own devices, unlocks 25 additional pet species, and fixes several peer-to-peer connection reliability issues. No new entitlements, no new permissions, no new data collection, no IAP changes.

## What's new in v5.4.0 → v5.5.0

**Cross-device pet sync (iCloud).** The virtual pet now syncs across the user's own Apple devices via their private iCloud (already entitled since v5.4). When the same iCloud account is used on, e.g., an iPhone and an iPad, both show the same pet, age and progress. Sync is best-effort and degrades gracefully to local-only when iCloud is unavailable — no account setup, no new prompt, no UI added. This is the user's own private data in their own iCloud; it is not collected by us and is never shared between different users.

**More pet species.** The pet-hatching system can now produce 25 additional species (hamster, fox, pig, owl, penguin, and more) on top of the original set. These are cosmetic variations of the existing virtual-pet feature — no new mechanics, no new data.

**Connection reliability fixes (peer-to-peer):**
- More robust peer-identity handling on the relay path (correct peer matching on reconnect).
- Fixed a rare race/deadlock in the transport readiness path that could leave a connection hung.
- Scoped the TLS certificate keychain queries correctly so local-Wi-Fi encryption negotiates reliably.
- Chat: Enter now sends a message; clearer feedback when a voice message fails to record.

**Pet UI polish.** The pet detail screen is now fully localized (English, Japanese, Korean, Simplified & Traditional Chinese) and carries VoiceOver accessibility labels.

## What didn't change

- End-to-end encryption (X3DH + Double Ratchet), local Wi-Fi + relay transports, SAS pairing — unchanged.
- File transfer, voice calls, screenshot mode — unchanged.
- Tip Jar IAPs (tip.small / tip.medium / tip.large), Privacy Manifest (no new data types collected), age rating 4+, supply regions — unchanged.
- No new entitlements, no new permission prompts.

## How to verify

1. Install v5.5.0. Open the app and tap the "Pet" tab — the virtual pet renders with localized labels.
2. Cross-device sync (optional; needs two devices on one iCloud account): feed/interact with the pet on one device, send the app to the background, then open the app on the second device — the same pet (name, age, progress) appears.
3. Peer connection: on two devices tap "New Room" / "Join Room" with the code (or connect over local Wi-Fi), then send a text message — expect normal delivery with no error indicator.
