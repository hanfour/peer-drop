# v4.0.1 — Remove Egg Stage Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Eliminate the `.egg` lifecycle stage from the Pet system in v4.0.1. Existing v3.x egg-stage pets auto-promote to baby. New users see an immediate "你獲得了 [species]！" welcome reveal instead of waiting for a hatch.

**Architecture:**
- **Data:** `PetLevel` keeps `rawValue 1` reserved for Codable backward compatibility but removes `.egg` case + adds custom decoder mapping `1 → .baby`. v3.x persisted JSON, peer payloads, and iCloud sync all decode safely.
- **Migration:** Silent — the decoder does the work. PetStore signals "show welcome screen" on next PetTab open if a transition just happened.
- **UI:** New `PetWelcomeView` modal screen with reveal animation. Triggered for both first-launch v4.0.1 users and v3.x → v4.0.1 migrators whose pets just hatched.
- **Cross-version:** `PetPayload` decoder also maps level=1 → .baby when receiving from v3.x peers.

**Tech Stack:**
- Swift 5.9, SwiftUI, Codable
- Existing: `PetLevel` enum, `PetState` struct, `PetStore`, `PetRendererV3`, `V4UpgradeOnboardingView`, `SpriteService`, Apple String Catalog (xcstrings)
- Tests: XCTest with NWListener loopback for integration

**Files affected (overview):**

- 18 source touchpoints: 11 Behavior files, PetState, EvolutionRequirement, SpriteAssetResolver, PetEngine, PetBehaviorController, PetDialogEngine
- 14 test files referencing `.egg`

```swift
import Foundation

/// Pet lifecycle stage. Persisted via Codable rawValue.
///
/// **rawValue 1 is reserved**: it was `.egg` in v3.x. v4.0.1 removed
/// the egg stage but the decoder maps any persisted/peer-sent
/// `rawValue 1` to `.baby` for backward compatibility (v3.x users on
/// upgrade, cross-version peer payloads, iCloud restore from older
/// devices).
enum PetLevel: Int, Codable, Comparable, CaseIterable {
    case baby = 2
    case adult = 3   // legacy "child" — rawValue stays 3 so persisted/network data migrates without a custom decoder
    case elder = 4

    init(from decoder: Decoder) throws {
        let raw = try decoder.singleValueContainer().decode(Int.self)
        // rawValue 1 = legacy .egg → .baby; any unknown future raw also falls to .baby
        self = PetLevel(rawValue: raw) ?? .baby
    }

    static func < (lhs: PetLevel, rhs: PetLevel) -> Bool {
        lhs.rawValue < rhs.rawValue
    }

    var displayName: String {
        switch self {
        case .baby: return "幼年"
        case .adult: return "成熟"
        case .elder: return "老年"
        }
    }

    var assetSlug: String {
        switch self {
        case .baby: return "baby"
        case .adult: return "adult"
        case .elder: return "elder"
        }
    }
}
```

```swift
import Foundation

/// Tracks whether the user has seen the v4.0.1 pet-welcome reveal.
/// First-launch users + v3.x→v4.0.1 migrators (whose .egg pet just
/// hatched) both should see the welcome on first PetTab open.
final class PetWelcomeFlag {
    private let key = "hasSeenPetWelcome_v4"
    private let defaults: UserDefaults

    init(defaults: UserDefaults = .standard) {
        self.defaults = defaults
    }

    var shouldShow: Bool {
        !defaults.bool(forKey: key)
    }

    func markSeen() {
        defaults.set(true, forKey: key)
    }
}
```

import SwiftUI

struct PetWelcomeView: View {
    let pet: PetState
    let onDismiss: () -> Void

    @State private var revealScale: CGFloat = 0.3
    @State private var revealOpacity: Double = 0
    @State private var celebrationOffset: CGFloat = -20

    var body: some View {
        ZStack {
            Color.black.opacity(0.85).ignoresSafeArea()

            VStack(spacing: 24) {
                Text("🎉")
                    .font(.system(size: 64))
                    .offset(y: celebrationOffset)
                    .animation(.easeInOut(duration: 1.4).repeatForever(autoreverses: true), value: celebrationOffset)

                Text("welcome.congrats")
                    .font(.title.bold())
                    .foregroundStyle(.white)

                PetSpriteView(pet: pet)   // existing renderer
                    .scaleEffect(revealScale)
                    .opacity(revealOpacity)
                    .frame(width: 200, height: 200)

                Text(pet.species.displayName)
                    .font(.title2.bold())
                    .foregroundStyle(.white)

                Text("welcome.subtitle")
                    .font(.body)
                    .foregroundStyle(.white.opacity(0.8))
                    .multilineTextAlignment(.center)
                    .padding(.horizontal, 32)

                Button {
                    onDismiss()
                } label: {
                    Text("welcome.cta")
                        .font(.headline)
                        .padding(.horizontal, 32)
                        .padding(.vertical, 12)
                }
                .buttonStyle(.borderedProminent)
                .tint(.white)
                .foregroundStyle(.black)
            }
        }
        .onAppear {
            withAnimation(.spring(response: 0.8, dampingFraction: 0.6).delay(0.3)) {
                revealScale = 1.0
                revealOpacity = 1
            }
            celebrationOffset = 20
        }
    }
}
